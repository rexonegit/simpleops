# 运维工作台 (simpleops) 项目技术栈

> 面向多云资源纳管、本地机房资产治理、配置管理与自动化运维的一体化工作平台。
>

---

## 一、整体架构

项目为 **前后端分离** 架构:

```
simpleops/
├── backend/     # Django + DRF 后端 API
├── web/         # Vue 3 + Element Plus 前端
├── scripts/     # 环境安装脚本
└── example/     # 示例截图
```

---

## 二、后端技术栈 (`backend/`)

### 1. Web 框架与核心库

| 类别 | 技术 / 库 | 说明 |
| :--- | :--- | :--- |
| Web 框架 | **Django** | 主框架 |
| REST API | **djangorestframework** (DRF) | 接口序列化 / 视图 |
| 鉴权 | djangorestframework-simplejwt | JWT Token 认证 |
| 过滤 | django-filter | 列表过滤 |
| 跨域 | django-cors-headers | 前后端分离 CORS |
| API 文档 | drf-yasg | Swagger / OpenAPI |
| 数据库驱动 | mysqlclient | MySQL 连接 |
| 环境变量 | python-dotenv | `.env` 配置管理 |
| 日期处理 | python-dateutil | 日期解析 |
| 图像处理 | Pillow | 图片处理 |
| Excel | openpyxl | Excel 导入导出 |
| 拼音 | pypinyin | 中文转拼音(检索/排序) |

### 2. 云平台 SDK (阿里云)

纳管阿里云多云资源,使用官方 `alibabacloud_*` SDK:

| SDK | 用途 |

| alibabacloud_ecs20140526 | ECS 云服务器 |
| alibabacloud_rds20140815 | RDS 云数据库 |
| alibabacloud_vpc20160428 | VPC 专有网络 |
| alibabacloud_slb20140515 | SLB 负载均衡 |
| alibabacloud_nas20170626 | NAS 文件存储 |
| alibabacloud_oss_v2 | OSS 对象存储 |
| alibabacloud_ims20190815 | IMS 身份管理 |
| alibabacloud_ram20150501 | RAM 访问控制 |
| alibabacloud_sl**s**20201230 | SLS 日志服务 |
| alibabacloud_alidns20150109 | DNS 解析 |
| alibabacloud_domain20180129 | 域名 |
| alibabacloud_waf_openapi20211001 | WAF Web 应用防火墙 |
| alibabacloud_tea / alibabacloud_tea_util / alibabacloud_tea_openapi | Tea SDK 公共依赖 |

### 3. 虚拟化 SDK

| SDK | 用途 |
| :--- | :--- |
| **pyvmomi** | VMware vCenter / ESXi 资源纳管 |

> ProxmoxVE 通过 HTTP API 自行对接(无独立 SDK 依赖)。

### 4. 后端应用模块

| Django App | 职责 |
| :--- | :--- |
| `aliyun/` | 阿里云资源同步与管理 |
| `cmdb/` | 配置管理数据库 (项目信息、IP 库、数据库台账) |
| `datacenter/` | 本地机房资产(VMware / ProxmoxVE / 物理机 / 网络设备) |
| `opsmgmt/` | 运维管理(告警登记等) |
| `apps/` | 公共应用配置 |
| `django_vue_admin/` | 后台管理与权限脚手架 |

### 5. 运行入口

- `manage.py` — Django 标准启动入口
- `clean_django.bat` — Windows 环境清理脚本
- `requirements.txt` — Python 依赖清单

## 五、技术栈速览

| 层级 | 核心技术 |

| **构建工具** | Rspack 1.5 |
